About Course

Front-end Development is everything involved in programming the user interface of a web application.Typically
it refers the hyper text Markup Launguage (HTML),Cascading Style Sheets (CSS) and Javasript portion of web
site production as opposed to the database or server-side programming.It encompasses everything from a building a
simple page of HTML text to creating a complex,responsive HTML5 websites designed to be accesssed via various
different browsers,devicesand screen sizes.
